Las condicionales en JavaScript son una forma de controlar el flujo de ejecución de tu programa. Permiten que tu código tome decisiones basadas en ciertas condiciones.

La estructura más básica de una condicional en JavaScript es la declaración if. Esta declaración toma una expresión booleana y ejecuta un bloque de código si esa expresión es verdadera.
